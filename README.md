# COMP3005 A3 Q1

## How to install and run

- Download all files in the repo manually or by cloning the repository:
> `git clone https://github.com/ramonacaprariu/COMP3005-A3.git`
>
> `cd COMP3005-A3/Assignments/Assignment_3`
- Run A3_Students.sql via the query tool in a newly constructed database in pgAdmin (and note your username and password)
- Create your virtual environment and install dependencies with the following:
> `python -m venv venv`
>
> `source venv/bin/activate` or on Windows, `venv\Scripts\activate`
>
> `pip install -r requirements.txt`
- Create a .env file based on the included .env.example, and replace with your own pgAdmin username and password
- Run A3_Students.py via your code editor or the command line:
> `python A3_Students.py`
